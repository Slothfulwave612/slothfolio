Contributions

Matplotlib

mplsoccer

Visualization Library

mplsoccer is a Python library for plotting soccer/football charts in Matplotlib and loading StatsBomb open-data. In mplsoccer, you can:

  • 1) plot football/soccer pitches on nine different pitch types
  • 2) plot radar charts
  • 3) plot Nightingale/pizza charts
  • 4) plot bumpy charts for showing changes over time
  • 5) plot arrows, heatmaps, hexbins, scatter, and (comet) lines
  • 6) load StatsBomb data as a tidy dataframe
  • 7) standardize pitch coordinates into a single format

Teammate: Andy Rowlinson

Matplotlib

soccerplots

Visualization Library

soccerplots is a Python library that can be used for making visualizations for football analytics. The main aim of soccerplots is to save time for analysts so they can focus more on the analysis rather than coding the visualizations from scratch. With soccerplots v1.0.0 the users can:

  • 1) Make their own defined theme for plotting radar charts.
  • 2) Use bumpy_chart module for plotting bump charts.

Machine Learning & Deep Learning

Player Fatigue and Performance

Freelance Project

The main aim was to build a model that can predict a Rating of Perceived Exertion (RPE) value, also called an internal load, given the external load parameters. Two seasons worth of athlete's data for each training and match-day session was given. External load parameter values were from the GPS devices, and the athlete answered the Internal load values through a wellness survey. I trained four models of which an Artificial Neural Network was the best performer.

Expected-Goals Model

Personal Project

The model measures the quality of a shot based on several variables such as shot angle, distance from goal etc. Analysts can use this model to evaluate the xG value for shots taken by the players. I have used Statsbomb open-data repository for training and testing the models, and have trained three models which I call - Basic Model, Intermediate Model and Advance Model.

  • 1) Basic Model: All basic features like distance, angle, shot-type & body-part.
  • 2) Intermediate Model: All basic features + pass-type, open-goal, under-pressure & deflected.
  • 3) Advance Model: All basic & intermediate model features + freeze-frame-data & goalkeeper-angle.

Deep Learning

Image-Super-Resolution

Personal Project

The goal of this project is to recover a high resolution image from a low resolution input. In order to accomplish this goal, I have used the super-resolution convolutional neural network (SRCNN) using Keras. This network was published in the paper, Image Super-Resolution Using Deep Convolutional Networks by Chao Dong, et al. in 2014. The SRCNN directly learns an end-to-end mapping between low and high-resolution images, as a result, we can use it to improve the image quality of low resolution images.

Web Scraping & Google Calendar

Footy-Events

Personal Project

This project, Footy Events, focusses on scrapping user's team fixtures and adding the fixture as an event in user's Google calendar. I am scrapping the content from skysports website. The program works like this:

  • 1) First, the user will feed in the team's or competition name.
  • 2) Then the program will automatically search for the relative query on Google, and go to the skysports website, hosting the required fixtures.
  • 3) The program will then scrape the fixtures from the website.
  • 4) Using Google-Calendar's API the program will finally add the fixtures as events in the user's calendar.
